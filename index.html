SHORT INTRO

A Glimpse of Reporter Pathway Analysis (RPA) Web Tool

Reporter Pathway Analysis Web Tool is a web implementation of paper written by Prof. Tunahan Çakır. The tool itself can be accessed through reporterpathways.org. Reporter Pathway Tools (also known as the RTools) allows the user to analysis their transcriptomics data using the method described in the paper. There are five tools available at the moment, Reporter Features (RF), Reporter Metabolite (RM), Reporter Pathway Analysis (RPAr), Reporter Pathway Analysis M2P (RPAmM2P), Reporter Pathway Analysis (RPAmG2P)

RTools also allow the user to create an account, thus it will be easier for the user to maintain their results history. RTools were written in R by a graduate student from the Bioinformatics and System Biology Department at Gebze Technical University İsa Yüksel, and the web itself has Python-Django as the backend and CSS Bootstrap for template.
